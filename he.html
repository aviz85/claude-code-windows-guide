התקנת Node.js

כ-2 דקות
Node.js היא סביבת הריצה שמפעילה את Claude Code. זו תוכנה חינמית שמשמשת מפתחים רבים.
התקנת Node.js בווינדוס

עקבו אחרי השלבים הבאים:

  1. פתחו את הדפדפן וחפשו "Node js" בגוגל
  2. לחצו על האתר הרשמי (nodejs.org)
  3. לחצו על הכפתור הירוק "LTS" להורדה (זו הגרסה היציבה)
  4. הריצו את הקובץ שהורדתם (שם הקובץ מסתיים ב-.msi)
  5. לחצו על "Next" באשף ההתקנה
  6. אשרו את הסכם הרישיון
  7. השאירו את תיקיית ההתקנה כברירת מחדל
  8. חשוב: אל תסמנו את האפשרות "Automatically install necessary tools" - השאירו אותה ריקה
  9. לחצו על "Install" והמתינו לסיום
  10. לחצו על "Finish"
איך יודעים שזה עבד?

פתחו Command Prompt והקלידו node --version. אם מופיע מספר (כמו v24.12.0), הכל בסדר!

2

התקנת Claude Code

כדקה
עכשיו נתקין את Claude Code עצמו באמצעות פקודה פשוטה ב-Command Prompt.
התקנת Claude Code דרך Command Prompt

עקבו אחרי השלבים הבאים:

  1. לחצו על תפריט התחל (אייקון החלון)
  2. הקלידו "cmd" או "Command Prompt"
  3. לחצו על Command Prompt כדי לפתוח אותו
  4. העתיקו והדביקו את הפקודה הזו, ולחצו Enter:
npm install -g @anthropic-ai/claude-code
מה הפקודה הזו עושה?

היא אומרת ל-Node.js להוריד ולהתקין את Claude Code מהאינטרנט. ה--g אומר "התקן גלובלית" כדי שתוכלו להשתמש בזה מכל מקום.

3

התקנת Git

כ-3 דקות
Git היא תוכנה חינמית שעוזרת לעקוב אחרי שינויים בקבצים. Claude Code צריך אותה כדי לעבוד כמו שצריך.
הורדה והתקנה של Git

עקבו אחרי השלבים הבאים:

  1. חפשו "git scm" בגוגל
  2. היכנסו לאתר הרשמי של Git (git-scm.com)
  3. לחצו על "Download for Windows"
  4. בחרו ב-"64-bit Git for Windows Setup" (Standalone Installer)
  5. הריצו את קובץ ההתקנה שהורדתם
  6. לחצו על "Next" ברוב האפשרויות (ברירות המחדל מצוינות)
אל תדאגו מהאפשרויות!

למתקין של Git יש הרבה מסכים עם אפשרויות טכניות. פשוט לחצו "Next" והשאירו את ברירות המחדל - הן עובדות מצוין עם Claude Code.

סיכום מהיר של אפשרויות Git (פשוט לחצו Next):

  1. עורך: Vim (ברירת מחדל) - זה בסדר
  2. ענף ברירת מחדל: תנו ל-Git להחליט
  3. סביבת PATH: "Git from command line and 3rd-party software" (מומלץ)
  4. SSH: השתמשו ב-OpenSSH המובנה
  5. HTTPS: השתמשו ב-Windows Secure Channel
  6. סיומי שורות: Checkout Windows-style, commit Unix-style
  7. טרמינל: השתמשו ב-MinTTY
  8. לבסוף, לחצו על "Install"
4

הגדרה והתחברות

כ-3 דקות
הגיע הזמן להפעיל את Claude Code ולחבר אותו לחשבון Claude שלכם. כאן מתחיל הקסם!
הפעלת Claude Code והתחברות

עקבו אחרי השלבים הבאים:

  1. צרו תיקייה חדשה לפרויקט (למשל, קליק ימני על שולחן העבודה ← חדש ← תיקייה ← קראו לה "test")
  2. פתחו את PowerShell או Command Prompt
  3. נווטו לתיקייה שלכם על ידי הקלדת cd Desktop\test (או הנתיב לתיקייה שלכם)
  4. הקלידו claude ולחצו Enter
  5. השתמשו בחיצים כדי לבחור ערכת נושא (כמו Dark mode) ולחצו Enter
  6. בחרו ב-"Claude account with subscription"
  7. ייפתח חלון דפדפן - התחברו לחשבון Claude שלכם
  8. לחצו על "Authorize" כדי לחבר את Claude Code
  9. תראו "You're all set up" - אפשר לסגור את הדפדפן
  10. חזרו לטרמינל, קראו את אפשרויות הפרטיות ולחצו Enter לאישור
צריכים מנוי?

אתם צריכים מנוי Claude Pro או Team. אם אין לכם, הדפדפן יציע לכם לשדרג. המנויים מתחילים מ-$20 לחודש.

מזל טוב!

Claude Code מוכן לשימוש! אתם יכולים להתחיל לשוחח עם Claude בטרמינל.

5

הגדרות פרטיות אופציונלי

כדקה
כברירת מחדל, השיחות שלכם עשויות לשמש לשיפור Claude. אם אתם מעדיפים יותר פרטיות, אפשר לשנות את ההגדרה הזו.
ניהול הגדרות פרטיות

כדי לבטל שימוש בנתונים לאימון:

  1. פתחו את הדפדפן וגלשו ל-claude.ai
  2. לחצו על אייקון הפרופיל שלכם בפינה העליונה
  3. לחצו על "Settings"
  4. עברו ללשונית "Privacy"
  5. כבו את האפשרות "Help improve Claude"
מה זה אומר?

כש-"Help improve Claude" מופעל, השיחות שלכם עשויות להיבדק על ידי Anthropic כדי לשפר את Claude. כיבוי האפשרות אומר שהנתונים שלכם נשארים פרטיים ולא ישמשו לאימון.
